Overview

a CSFDocument subclass. Presentation for platform runtime document, this class is used as platform depentdent class and used mainly for file write/read process. For the platform-indepentdent data model, see CSFCetaceaAbstractSharedDocument

On iOS:

CSFDocument = UIDocument CSFCetaceaNativeSharedDocument = CSFCetaceaUIDocument so CSFCetaceaNativeSharedDocument is a subclass of UIDocumet

On OSX:

CSFDocument = NSDocument CSFCetaceaNativeSharedDocument = CSFCetaceaNSDocument so CSFCetaceaNativeSharedDocument is a subclass of NSDocument
